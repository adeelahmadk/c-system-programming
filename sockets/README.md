# POSIX Socket API

## API functions used

- socket
- bind
- listen
- accept
- recv
- sendfile
- bzero
- htons
- inet_pton
- connect

## Programs

| Program | Description |
| ------- | ----------- |
| `httpsrv` | A server program that listens for HTTP GET requests and return an HTML page. |
| `httpclient` | A client program that sends a GET request to a web server and prints the response. |
|  |  |
